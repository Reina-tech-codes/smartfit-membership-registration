# SmartFit Membership Registration System

## Overview
A Python console application that simulates a fitness center registration system.

## Features
- Age-based membership recommendations
- Medical condition validation
- Medical clearance checks
- Basic and Premium membership plans
- Personal training option
- Youth discount eligibility
- Input validation using while loops
- Confirmation system

## Concepts Used
- Variables
- if/elif/else statements
- Nested conditionals
- Logical operators
- while loops
- Dictionaries
- User input
